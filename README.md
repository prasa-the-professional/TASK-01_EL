# Linux Basics Command Reference

This repository provides a concise yet expanded conceptual reference for essential Linux commands. It is designed for beginners, students, and interview preparation, focusing on **why and when** each command is used rather than just syntax.

## Purpose

The goal of this repository is to help users:

* Understand core Linux commands conceptually
* Navigate and manage the filesystem confidently
* Work efficiently with files, permissions, and processes
* Build a strong foundation for system administration and DevOps roles

## Covered Topics

### Directory Navigation

Commands to understand filesystem structure and move efficiently between directories.

* pwd
* ls, ls -l, ls -a
* cd, cd .., cd ~
* Root and home directories

### File and Directory Management

Commands used to create, move, copy, and delete files and directories safely.

* mkdir, mkdir -p
* touch
* cp, mv
* rm, rm -r, rmdir

### Viewing and Editing Files

Commands to inspect and edit files directly from the terminal.

* cat
* less
* head, tail, tail -f
* vim

### Ownership and Permissions

Commands to control access and security of files.

* ls -l
* chown
* chmod

### Search and Utilities

Commands for searching, filtering, and analyzing text data.

* find
* grep
* wc
* sort

### System Monitoring

Commands to inspect system health and resource usage.

* top
* ps aux
* df -h
* du -sh
* free -m
* uptime

### Documentation and Help

Commands to explore official documentation and command history.

* man
* command --help
* history
* clear
